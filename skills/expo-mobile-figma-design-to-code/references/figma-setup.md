# Figma Setup

This skill assumes the user already has Figma MCP working.

## Expected setup

- Figma MCP added to Codex
- OAuth login completed
- Codex restarted after enabling the MCP

## Minimum verification

- A Figma URL or node ID is available
- `get_design_context` can return node data
- `get_screenshot` can return a matching preview

If setup is missing, stop and fix setup before attempting implementation.
