# Tokens And Composition

## Token buckets

- Color: `bg`, `fg`, `border`, `accent`, `success`, `warning`, `danger`
- Space: `xs`, `sm`, `md`, `lg`, `xl`
- Radius: `sm`, `md`, `lg`, `xl`, `full`
- Typography: display, heading, body, label, caption
- Elevation: 0 to 3

## Composition order

1. Tokens
2. Primitives
3. Feature components
4. Screens

## Primitive examples

- `AppText`
- `AppButton`
- `AppInput`
- `AppCard`
- `AppSheet`

## Rule

Tailwind or NativeWind classes are an implementation mechanism. The design system is the combination of tokens, primitives, states, and naming.
