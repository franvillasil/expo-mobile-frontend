# Privacy Policy

`expo-mobile-frontend` is a local Codex plugin. It does not ship a hosted backend, analytics collector, or remote storage layer.

## What the plugin does

- Provides local `SKILL.md` guidance files
- Optionally points users to external tools such as Figma MCP
- Includes a local installer that updates the user's personal marketplace file

## Data handling

- The plugin itself does not collect, transmit, or store user data
- Any data handled by Codex, GitHub, Figma, or other third-party services is governed by those services
- Optional integrations are user-initiated and not bundled with remote credentials
